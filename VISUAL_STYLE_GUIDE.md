## ✅ Color Contrast Standards (WCAG Compliance)

### AAA Level (Enhanced - 7:1+)
```
✅ #0F1419 text on #FFFFFF (15.8:1)
✅ #FFFFFF text on #0084FF (6.2:1)
✅ #FFFFFF text on #00C896 (5.1:1)
✅ #0F1419 text on #E8F0FF (13.2:1)
```

### AA Level (Standard - 4.5:1+)
```
✅ #6B7280 text on #FFFFFF (5.5:1)
✅ #FFFFFF text on #003366 (8.1:1)
✅ #FFFFFF text on #FF6B4A (4.5:1)
```

### Avoid (Insufficient Contrast)
```
❌ #9CA3AF on #FFFFFF (3.2:1)
❌ Light text on light backgrounds
❌ Low opacity combinations
```

## 🎬 Animation Guidelines - Professional

### Transition Timings
```
Quick:     100ms  (button press, hover state)
Standard:  300ms  (screen transitions, color changes)
Slow:      500ms  (enter/exit animations)
```

### Easing Functions
```
Entrance:   ease-out 300ms
Exit:       ease-in 200ms
Hover:      ease-out 150ms
Focus:      ease-out 200ms
Standard:   cubic-bezier(0.4, 0, 0.2, 1)
```

### Animations
```
Button Press:       Scale 0.95-1.0 (100ms)
Card Hover:         Elevation + scale 1.02 (200ms)
Screen Entrance:    Fade + slide 300ms
Ripple Effect:      400ms radial expansion
Loading:            Rotation 1s linear infinite
```

- Font sizes: Normal

- Layout: Optimized single column

### Large Phones (6.7" - 7")
- Font sizes: +1-2sp
- Spacing: 20dp
- Cards: Can support 2-column on landscape
- Layout: More breathing room

### Tablets (10"+)
- Font sizes: +2-3sp
- Spacing: 24-32dp
- Cards: 2-3 column layout possible
- Sidebar: Navigation on side
- Layout: Premium whitespace

### Landscape Orientation
- Reduce vertical height requirements
- Increase horizontal spacing
- Side-by-side layouts preferred
- Maintain touch targets (48dp minimum)
