## 📞 Support Information

### If You Need to Customize:
1. Refer to `DESIGN_SYSTEM.md` for all color codes
2. Check `IMPLEMENTATION_GUIDE.md` for how-to steps
3. Update `values/colors.xml` for color changes
4. Update `values/themes.xml` for style changes

### If Something Breaks:
1. Run `./gradlew clean build` to clear cache
2. Check `get_errors` for compilation issues
3. Verify all drawables exist in correct folders
4. Ensure Material3 dependencies are updated

### If You Want to Change Colors:
1. Open `app/src/main/res/values/colors.xml`
2. Find the color you want to change
3. Update the hex value
4. Rebuild the app
5. Changes apply everywhere automatically!
